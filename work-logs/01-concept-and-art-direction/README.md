## 4. Key Decisions

| Decision | Rationale |
|----------|-----------|
| Title: "Within Parameters" | Thematic (AI status report language), no existing game conflicts |
| Single protagonist | Scope control — 3 POV characters too ambitious for portfolio piece |
| NightCafe over NB2 for exploration | NB2 had severe style convergence; NightCafe free tier produced usable variety |
| DreamShaper XL Lightning + Hyperreal | Best balance of detail density, atmospheric mood, and compositing compatibility |
| NB2 reserved for 4K production | Use NightCafe for drafts/selection, NB2 for final high-res assets |
